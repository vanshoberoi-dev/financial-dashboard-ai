# Changelog

All notable changes to Finora — Indian Finance Dashboard will be documented in this file.

## [1.0.0] - 2026-03-08

### Features

- **Portfolio Dashboard** — Unified view of all investments with real-time valuation, asset allocation breakdown, and performance tracking across equity, debt, gold, and real estate.
- **Mutual Fund Central Connect** — Integration with MF Central/CAMS/KFintech for auto-import of mutual fund folios, NAV tracking, SIP management, and XIRR-based return calculation.
- **AI Financial Advisor** — GPT-powered advisory engine providing personalized investment recommendations, risk profiling, goal-based planning, and tax-efficient portfolio rebalancing suggestions.
- **Stock Tracker** — Real-time NSE/BSE stock tracking with watchlists, technical indicators, price alerts, intraday charts, and fundamental analysis powered by market data APIs.
- **Insurance Comparator** — Side-by-side comparison of term life, health, and motor insurance plans across major Indian insurers with premium calculators and claim settlement ratio analysis.
- **Tax Calculator** — Comprehensive Indian income tax calculator supporting Old and New regimes, Section 80C/80D deductions, HRA exemption, capital gains tax, and advance tax estimation for FY 2025-26.

### Notes

- Initial release of Finora Indian Finance Dashboard
- Built with Next.js 14, React 18, TailwindCSS 3, Recharts, and Radix UI
- Backend powered by FastAPI with PostgreSQL
